# **App Name**: Ninetech Portfolio

## Core Features:

- Case Study Showcase: Display case studies with descriptions, images, and links to deployed projects or GitHub repositories.
- Contact Form: A simple contact form for potential clients to send messages directly. This submits to your email address.
- Services Overview: Clearly present your expertise, services offered, and technologies used. Include brief descriptions and visuals where appropriate.
- Call-to-Action Buttons: Include clear call-to-action buttons (e.g., 'Get a Quote,' 'View Project') strategically placed throughout the site.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and technological expertise, inspired by the depth of embedded systems and the night sky.
- Background color: Light Gray (#F0F2F5) to ensure readability and a clean, modern aesthetic.
- Accent color: Electric Blue (#7DF9FF) to highlight interactive elements and CTAs, creating a sense of energy and innovation. The electric blue adds a futuristic/energetic flair.
- Clean, sans-serif fonts to maintain a modern and professional look.
- Use a set of consistent, minimalist icons to represent technologies, services, and expertise.
- Employ a grid-based layout for a structured and responsive design.
- Subtle animations, like smooth scrolling and hover effects, to add polish and interactivity without overwhelming the user.
